## 📺 Sinopse do Projeto
Em um mundo de códigos e desafios, surge o *Jogo Quiz*. Uma experiência interativa onde o espectador (você!) se torna o protagonista. Teste seus conhecimentos em uma interface dinâmica onde cada escolha reflete o seu destino no jogo.

### 🎭 Gênero: Interativo, Educativo, Lógica.

---

## 🎬 Trailer (Funcionalidades)
*   *[EP 01] Avanço Automático:* Acerte a questão e seja "teletransportado" para a próxima cena sem cliques extras.
*   *[EP 02] Feedback Cromático:* 
    *   🔴 *Erro:* A opção brilha em vermelho, mas a verdade (verde) é revelada.
    *   🟢 *Acerto:* O caminho fica verde para a vitória.
*   *[EP 03] Interface Fluida:* Manipulação de DOM via JQuery para transições suaves.
