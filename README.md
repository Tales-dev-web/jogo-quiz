Desenvolvi um jogo de perguntas e resposta com o objetivo de colocar em prática conceitos de JavaScrip e JQuery.

<b>Como funciona:</b>

✔️ Ao acertar a pergunta, o jogo avança automaticamente para próxima pergunta, e passara para a proxima pergunta.

✖️ Ao errar, a alternativa escolhida fica vermelha, enquanto a resposta correta fica verde, e passara para a proxima pergunta.

Seguimos evoluindo! 💻 📚
